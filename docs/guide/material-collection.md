# 素材收集

Fredica 支持从多种来源导入视频素材，并统一管理在素材库中。

## 素材库概览

素材库（Library）是 Fredica 的核心模块，所有导入的视频都在此管理。

主要功能：
- 视频列表展示（缩略图、标题、时长、来源、处理状态）
- 按分类筛选
- 按处理状态筛选（全部 / 待处理 / 已下载 / 已转录）
- 删除素材
- 为视频打分类标签

## 从 B 站收藏夹导入

这是目前最完整的导入方式。

**操作步骤：**

1. 进入 **添加资源 → B 站 → 收藏夹**
2. 输入收藏夹 ID，点击查询
3. 系统会列出该收藏夹下的全部视频（支持分页浏览）
4. 勾选需要导入的视频（可全选）
5. 可选择关联到已有分类
6. 点击 **导入** 完成操作

**导入内容：**

导入时会保存以下元数据（不下载实际视频文件）：

| 字段 | 说明 |
|------|------|
| 标题 | 视频标题 |
| 封面 | 封面图片 URL（本地代理缓存） |
| 时长 | 视频时长（秒） |
| UP 主 | 视频作者 |
| 播放数 | 播放次数 |
| 收藏数 | 收藏次数 |
| 来源 ID | B 站 BV 号（如 BV1NK4y1V7M5） |

::: info 重复导入
相同 BV 号的视频不会重复导入，系统会自动去重。
:::

## 素材分类管理

分类是组织素材的基本单位，支持多对多关联（一个视频可属于多个分类）。

**创建分类：**
1. 在素材库页面点击 **新建分类**
2. 输入分类名称（必填）和描述（可选）
3. 确认创建

**为视频添加分类：**
1. 在视频卡片上点击编辑（分类按钮）
2. 勾选一个或多个分类
3. 保存

**删除分类：**

删除分类会同时移除该分类与所有视频的关联，但不会删除视频本身。

## 处理状态说明

每个素材视频有一个处理流水线状态（`pipeline_status`），表示当前进度：

| 状态 | 说明 |
|------|------|
| `pending` | 刚导入，仅有元数据，未下载视频文件 |
| `downloaded` | 视频文件已下载到本地 <Badge type="warning" text="尚待开发" /> |
| `transcribed` | 已完成语音识别，字幕文件已生成 <Badge type="warning" text="尚待开发" /> |
| `analyzed` | 已完成 AI 内容分析 <Badge type="warning" text="尚待开发" /> |
