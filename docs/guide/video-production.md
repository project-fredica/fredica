# 视频制作与导出

<Badge type="warning" text="尚待开发" />

::: warning 尚待开发
本章节介绍的功能均处于规划阶段，当前版本尚未实现。
:::

完成文稿编辑后，Fredica 可根据文稿与素材的时间标记，自动匹配视频片段，生成初步剪辑版本。

## 自动剪辑

系统根据文稿中的时间标记，自动从对应素材中截取视频片段，并按文稿顺序拼接，生成初剪版本。

**前提条件：**
1. 相关素材已完成下载（状态: `downloaded`）
2. 文稿已完成编辑，且时间标记与素材正确关联

**输出格式：**
- 导出为 mp4 文件（拼接后的初剪视频）
- 或直接在应用内预览

## 导出为 DaVinci Resolve 工程

对于需要进一步精修的项目，可以导出为 DaVinci Resolve（达芬奇）工程文件（`.drp`）。

**包含内容：**
- 原始视频素材（或引用路径）
- 文稿对应的剪辑时间线
- 字幕轨道

::: tip 为什么选择 DaVinci Resolve？
DaVinci Resolve 提供免费的专业级视频编辑功能，支持颜色分级、音频混音等专业处理，是进一步精修的理想选择。
:::

## 字幕烧录 <Badge type="warning" text="尚待开发" />

将生成的字幕文件（SRT）烧录到视频中，支持自定义字体、大小、位置和样式。

## 封面生成 <Badge type="warning" text="尚待开发" />

根据视频内容自动推荐封面截帧，支持添加标题文字和简单样式。
