# 其他功能

本章介绍 Fredica 的辅助功能，包括字幕翻译、配音、模板以及协作功能。

## 字幕翻译 <Badge type="warning" text="尚待开发" />

::: warning 尚待开发
字幕翻译功能尚未实现。
:::

对视频的字幕文件进行翻译，支持将中文字幕翻译为英文、日文、韩文等多种语言。

**预期支持语言：**
- 中文 ↔ 英文
- 中文 ↔ 日文
- 中文 ↔ 韩文
- 更多语言对（通过 AI 翻译接口）

## 文本转语音配音 <Badge type="warning" text="尚待开发" />

::: warning 尚待开发
TTS（文本转语音）配音功能尚未实现。
:::

根据文稿内容生成合成语音，用于视频配音。

**预期功能：**
- 多种音色选择（男声/女声、不同风格）
- 语速、音调可调节
- 支持为不同段落设置不同音色

::: warning 尚待开发
模板功能尚未实现。

## 应用设置

以下设置功能当前已可用：

### 服务器配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 服务端口 | API 服务器监听端口 | `7631` |
| 数据目录 | 本地数据存储路径 | `.data/` |

### 行为设置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 自动启动 | 系统启动时自动运行 Fredica | 否 |
| 最小化到托盘 | 启动时最小化到系统托盘 | 否 |
| 启动时打开浏览器 | 服务启动后自动打开 Web UI | 是 |

### 外观

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 主题 | 亮色 / 暗色 / 跟随系统 | 跟随系统 |
| 语言 | 界面语言 | 简体中文 |

### 网络代理

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用代理 | 是否通过代理访问外部网络 | 否 |
| 代理地址 | HTTP 代理 URL | — |
| RSSHub 地址 | 自定义 RSSHub 实例 URL | — |
