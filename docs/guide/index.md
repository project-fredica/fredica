# 用户文档

欢迎使用 Fredica！本文档面向所有使用 Fredica 进行视频创作的用户。

## 适用场景

Fredica 适合以下使用场景：

- **短视频创作者** — 快速整理和分析素材，辅助脚本创作
- **课程录制者** — 批量处理视频，自动生成字幕和文稿
- **企业宣传团队** — 多人协作，统一管理素材库
