文稿辅助生成 ​

尚待开发

尚待开发

本章节介绍的功能均处于规划阶段，当前版本尚未实现。

文稿辅助生成模块是 Fredica 的核心 AI 能力之一，能够根据用户选取的素材内容，自动分析语音和文字信息，生成多种模式的视频脚本草稿。

工作原理 ​

文稿生成依赖已完成转录（transcribed 状态）的素材，流程如下：

选取素材 → AI 分析内容 → 生成脚本草稿 → 用户编辑修改 → 确认文稿
1

生成模式 ​

摘要精简模式 ​

从多段视频内容中提取核心要点，生成简洁的视频脚本。适合将长视频压缩为短视频的场景。

重新讲述模式 ​

保留原视频的主要内容，用更流畅、更适合目标平台的语言风格重新组织表达。

主题聚焦模式 ​

用户输入一个主题或问题，AI 从所选素材中筛选相关内容，围绕该主题生成结构化脚本。

对比分析模式 ​

对比多个素材的观点或数据，生成分析类的视频文案。

文稿编辑器 ​

生成草稿后，用户在文稿编辑器中进行编辑：

  • 时间轴关联：每段文字与原视频片段的时间戳关联，方便定位
  • 原文对照：侧边栏显示对应的原始转录文字
  • 实时预览：根据当前文稿预览匹配后的视频片段
  • 版本历史：保存多个编辑版本

注意事项 ​

AI 模型配置

文稿生成功能需要配置 AI 服务 API Key（OpenAI 或兼容接口）。可在设置页面中配置。

最佳实践

建议在生成前先检查转录质量，错误的转录会直接影响文稿生成效果。
